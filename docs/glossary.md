# Glossary

- Canonical facts: Imported WHOOP records stored as provider-derived truth, not edited by user preferences.
- Goal profile: A time-versioned weekly target set for HR zones and session counts.
- Sync run: One audited attempt to fetch and import provider data.
- ISO week: A Monday-start week using ISO calendar rules.
- Zone progress: Actual HR-zone training volume compared with the active goal profile for that week.
- Remaining volume: Target minutes minus completed minutes, never shown below zero.
- Current week: The ISO week containing today in the app's configured time context.
- 6-month report: A long-range summary built from the shared backend report dataset.
- Cardio session: A workout classified as endurance/cardio by deterministic sport rules or allowed manual overlay.
- Strength session: A workout classified as strength/resistance training by deterministic sport rules or allowed manual overlay.
